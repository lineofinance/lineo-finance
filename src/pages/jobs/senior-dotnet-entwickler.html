Senior .NET Entwickler (m/w/d)

Remote / München Vollzeit Engineering

Über die Rolle

Als Senior .NET Entwickler bei Lineo Finance spielen Sie eine Schlüsselrolle bei der Entwicklung unserer hochperformanten Backend-Lösungen für die automatisierte Wertpapierbuchhaltung. Sie arbeiten an der Spitze der Fintech-Innovation und gestalten Lösungen, die von Tausenden von Nutzern täglich verwendet werden.

Ihre Aufgaben

  • Entwicklung und Wartung von skalierbaren .NET Core/6+ Microservices
  • Design und Implementierung von RESTful APIs und Event-Driven Architectures
  • Optimierung der Performance und Skalierbarkeit unserer Systeme
  • Integration mit externen Finanz-APIs und Broker-Schnittstellen
  • Code Reviews und Mentoring von Junior-Entwicklern
  • Zusammenarbeit mit Product Managern und UX-Designern
  • Teilnahme an agilen Entwicklungsprozessen (Scrum)
  • Kontinuierliche Verbesserung unserer CI/CD-Pipeline

Ihr Profil

  • Mindestens 5 Jahre Erfahrung in der .NET-Entwicklung
  • Expertenkenntnisse in C#, .NET Core/6+, ASP.NET Core
  • Erfahrung mit Microservices-Architekturen und Domain-Driven Design
  • Fundierte Kenntnisse in SQL (vorzugsweise PostgreSQL) und NoSQL-Datenbanken
  • Erfahrung mit Message Queuing (RabbitMQ, Azure Service Bus)
  • Vertrautheit mit Cloud-Plattformen (Azure, AWS)
  • Kenntnisse in Docker, Kubernetes von Vorteil
  • Erfahrung mit agilen Entwicklungsmethoden
  • Sehr gute Deutschkenntnisse in Wort und Schrift
  • Teamgeist und ausgeprägte Kommunikationsfähigkeiten

Nice to have

  • Erfahrung im Fintech-Bereich oder mit Finanzdaten
  • Kenntnisse in Event Sourcing und CQRS
  • Erfahrung mit Blazor oder anderen Frontend-Technologien
  • Azure DevOps oder GitHub Actions Erfahrung
  • Zertifizierungen (Azure, .NET)

Was wir bieten

  • Vollständig remote oder hybrid aus unserem Münchner Büro
  • Attraktives Gehalt zwischen 70.000 - 95.000 EUR (je nach Erfahrung)
  • 30 Tage Urlaub + flexible Arbeitszeiten
  • Jährliches Weiterbildungsbudget von 3.000 EUR
  • Modernste Hardware Ihrer Wahl (MacBook Pro, Dell XPS, etc.)
  • Betriebliche Altersvorsorge
  • Gesundheitsbudget für Sport und Wellness
  • Quartalsweise Team-Events und Offsites
  • Die Chance, die Zukunft der Finanzbranche mitzugestalten
  • Flache Hierarchien und kurze Entscheidungswege

Unser Tech Stack

.NET 8 C# ASP.NET Core PostgreSQL Redis Docker Kubernetes Azure RabbitMQ Git

Der Bewerbungsprozess

  1. Bewerbung einreichen: Senden Sie uns Ihren Lebenslauf und ein kurzes Anschreiben
  2. Erstes Kennenlernen (30 Min): Video-Call mit unserem Recruiting-Team
  3. Technisches Interview (60 Min): Gespräch mit unserem Engineering Lead
  4. Coding Challenge: Eine kleine Programmieraufgabe (max. 3 Stunden)
  5. Team-Gespräch (45 Min): Kennenlernen des Teams und der Kultur
  6. Angebot: Bei gegenseitigem Interesse erhalten Sie unser Angebot
